# text2banner

A lightweight web app to generate ASCII/Unicode banners for CLI tools and GitHub READMEs. No installation needed — just visit the page, type, and copy.

**[Try it live →](https://YOUR_USERNAME.github.io/text2banner/)**

## Features

- Live preview as you type
- 6 figlet font styles (Big, Standard, Slant, Shadow, Small, Banner)
- 3 border styles (Unicode box, ASCII, none)
- Subtitle support
- Left / center / right alignment
- One-click copy to clipboard

## Contributing

Clone the repo, open `index.html` in a browser (or run `python -m http.server 8080`). No build step, no npm install.

Pull requests welcome — see open issues for ideas.
